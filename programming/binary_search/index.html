    4.寻找两个正序数组的中位数#

    题目: 给定两个大小分别为 m 和 n 的正序（从小到大）数组 nums1 和 nums2。请你找出并返回这两个正序数组的 中位数。算法的时间复杂度应该为 O(log (m+n)) 。

    思路: 划分法。记索引 \(i\) 划分 nums1, 索引 \(j\) 划分 nums2, 则满足

    \[ len(\text{nums1}[0:i]) + len(\text{nums2}[0:j]) = len(\text{nums1}[i:]) + len(\text{nums2}[j:]) \]

    def findMedianSortedArrays(self, nums1: List[int], nums2: List[int]) -> float:

    # assure len(nums1) <= len(nums2)
    if len(num1) > len(num2):
        nums1, nums2 = nums2, nums1
    m, n = len(nums1), len(nums2)

    # 边界处理
    if len(nums1) == 0:
        if len(nums2) % 2 == 0:
            return 0.5 * (nums2[n // 2 - 1] + nums2[n // 2])
        else:
            return nums2[n // 2]

    # 对 nums1 进行二分查找, 左闭右开区间 [left, right)
    # i: 为指向 nums1 中元素的索引;
    # j: 为指向 nums2 中元素的索引;
    left, right = 0, m + 1
    while left < right:
        i = (left + right) // 2
        j = (m + n + 1) // 2 - i

        # 异常处理，防止使用索引获取值时导致溢出
        tl = float('-inf') if i == 0 else nums1[i - 1]
        bl = float('-inf') if j == 0 else nums2[j - 1]
        tr = float('inf') if i == m else nums1[i]
        br = float('inf') if j == n else nums2[j]

        if tr < bl:
            left = i + 1  # +1
        elif tl > br:
            right = i
        else:
            break  # 跳出循环

    if (m + n) % 2 == 1:  # odd
        return max(tl, bl)
    else:  # even
        return 0.5 * (max(tl, bl) + min(tr, br))

    时间复杂度: \(O\left(\log\left(\min(m, n)\right)\right)\), 空间复杂度: \(O(1)\)

    33.搜索旋转排序数组#

    题目: 在传递给函数之前，nums 在预先未知的某个下标 k（0 <= k < nums.length）上进行了 旋转，使数组变为 [nums[k], nums[k+1], ..., nums[n-1], nums[0], nums[1], ..., nums[k-1]]（下标 从 0 开始 计数）。例如， [0,1,2,4,5,6,7] 在下标 3 处经旋转后可能变为 [4,5,6,7,0,1,2] 。给你 旋转后 的数组 nums 和一个整数 target ，如果 nums 中存在这个目标值 target ，则返回它的下标，否则返回 -1 。 你必须设计一个时间复杂度为 O(logn) 的算法解决此问题。

    思路:

    • mid 指针将数组切分为两部分, 必定一部分有序, 另一部分是非有序的。 如图所示是 [0, mid] 有序;
    • 如果 target 落入有序部分, 那么直接可以使用二分查找.

    方法1: 先二分查找旋转点，再根据旋转点判定 target 落入的有序区间, 最后在有序区间二分查找 target;

    方法2: 直接判定有序区间, 在有序区间中二分查找.

    class Solution:
    def search(self, nums, target):
        # 寻找旋转点 (二分查找)
        left, right = 0, len(nums)
        while left < right:
            mid = (left + right) // 2
            if nums[mid] == target:
                return mid
            if nums[mid] > nums[left]:
                left = mid
            else:
                right = mid

        # 判定 target 在旋转点的左半部分还是右半部分, 并设定查找区间
        if nums[0] <= target <= nums[mid]:
            left, right = 0, mid
        else:
            left, right = mid + 1, len(nums)

        # 寻找 target (二分查找)
        while left < right:
            mid = (left + right) // 2
            if nums[mid] < target:
                left = mid + 1
            elif nums[mid] > target:
                right = mid
            else:
                return mid
        return -1

    class Solution:
    def search(self, nums: List[int], target: int) -> int:
        left, right = 0, len(nums)
        while left < right:
            mid = (left + right) // 2
            if nums[mid] == target:
                return mid

            # [0, mid] 有序
            if nums[0] <= nums[mid]:
                # target 在 [0, mid) 中
                if nums[0] <= target < nums[mid]:
                    right = mid
                else:
                    left = mid + 1
            # [mid, end] 有序
            else:
                if nums[mid] < target <= nums[-1]:
                    left = mid + 1
                else:
                    right = mid
        return -1

    34.在排序数组中查找元素的第一个和最后一个位置#

    题目: 给你一个按照非递减顺序排列的整数数组 nums，和一个目标值 target。请你找出给定目标值在数组中的开始位置和结束位置。 如果数组中不存在目标值 target，返回 [-1, -1]。 你必须设计并实现时间复杂度为 O(log n) 的算法解决此问题。

    class Solution:
    def searchRange(self, nums: List[int], target: int) -> List[int]:
        def lower_bound(nums, target):
            left, right = 0, len(nums)
            while left < right:
                mid = (left + right) // 2
                if nums[mid] < target:
                    left = mid + 1
                else:
                    right = mid
            return left

        lower = lower_bound(nums, target)
        if (not nums) or (lower > len(nums) - 1) or (nums[lower] != target):
            return [-1, -1]
        upper = lower_bound(nums, target + 1) - 1
        return [lower, upper]

    240.搜索二维矩阵II#

    题目: 编写一个高效的算法来搜索 m x n 矩阵 matrix 中的一个目标值 target 。该矩阵具有以下特性：(1) 每行的元素从左到右升序排列。(2) 每列的元素从上到下升序排列。

    思路: 该题有两种高效的思路, 第一种是逐行二分查找, 时间复杂度为 \(O(m\log n)\); 第二种是 Z 字形查找, 即从右上角开始查找 \(num[i][j] > target\) 向左移动, \(nums[i][j] < target\) 向下移动, 时间复杂度为 \(O(m+n)\), 这里记录第二种写法。

    class Solution:
    def searchMatrix(self, matrix: List[List[int]], target: int) -> bool:
        m, n = len(matrix), len(matrix[0])
        i, j = 0, n - 1
        while i < m and j >= 0:
            if matrix[i][j] > target:
                j -= 1
            elif matrix[i][j] < target:
                i += 1
            else:
                return True
        return False
