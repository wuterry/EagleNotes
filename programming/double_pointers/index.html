    11.盛最多水的容器#

    题目: 给定一个长度为 n 的整数数组 height 。有 n 条垂线，第 i 条线的两个端点是 (i, 0) 和 (i, height[i]) 。 找出其中的两条线，使得它们与 x 轴共同构成的容器可以容纳最多的水。 返回容器可以储存的最大水量。

    class Solution:
    def maxArea(self, height: List[int]) -> int:
        """
        1. 双指针法
        2. 移动 height 较小的指针
        """
        left, right = 0, len(height) - 1
        max_area = 0
        while left < right:
            area = min(height[left], height[right]) * (right - left)
            if area > max_area:
                max_area = area
            if height[left] < height[right]:
                left += 1
            else:
                right -= 1
        return max_area

    15.三数之和#

    题目: 给你一个整数数组 nums ，判断是否存在三元组 [nums[i], nums[j], nums[k]] 满足 i != j、i != k 且 j != k ，同时还满足 nums[i] + nums[j] + nums[k] == 0。请你返回所有和为 0 且不重复的三元组。注意：答案中不可以包含重复的三元组。

    Tips: 本题需要注意的是当 满足三数和为0之后 的去重。

    class Solution:
    def threeSum(self, nums: List[int]) -> List[List[int]]:
        # 升序排列
        nums.sort()   # or 'nums = sorted(nums)'

        results = []
        for i in range(len(nums)):
            if i > 0 and nums[i - 1] == nums[i]:
                continue

            # 双指针缩小区间
            left, right = i + 1, len(nums) - 1
            while left < right:
                sum_ = nums[i] + nums[left] + nums[right]  # 三数和
                if sum_ > 0:
                    right -= 1
                elif sum_ < 0:
                    left += 1
                else:
                    results.append([nums[i], nums[left], nums[right]])
                    while left < right and nums[left] == nums[left + 1]:
                        left += 1
                    while left < right and nums[right] == nums[right - 1]:
                        right -= 1
                    left += 1
                    right -= 1

        return results

    18.四数之和#

    题目: 给你一个由 n 个整数组成的数组 nums ，和一个目标值 target 。请你找出并返回满足下述全部条件且不重复的四元组 [nums[a], nums[b], nums[c], nums[d]] （若两个四元组元素一一对应，则认为两个四元组重复):

    • 0 <= a, b, c, d < n
    • a、b、c 和 d 互不相同
    • nums[a] + nums[b] + nums[c] + nums[d] == target
    • 你可以按 任意顺序 返回答案 。

    思路: 同三数之和的思路, 主要关注去重:

    • 确定第 1 个数 \(nums[i]\) 之后:
      • 如果 \(nums[i] + nums[-1] + nums[-2] + nums[-3] < target\), 那么剩下的三个数无论如何取值, 四数之和一定小于 target, 进入下一轮循环 \(nums[i+1]\)；
      • 如果 \(nums[i] + nums[i + 1] + nums[i + 2] + nums[i + 3] > target\), 那么剩下的三个数无论如何取值，四数之和一定大于 target, 退出第一重循环.
    • 确定前两个数 \(nums[i]\) 和 \(nums[j]\) 之后:
      • 如果 \(nums[i] + nums[j] + nums[-1] + nums[-2] < target\), 那么剩下的两个数无论如何取值, 四数之和一定小于 target, 进入下一轮循环 \(nums[j+1]\);
      • 如果 \(nums[i] + nums[j] + nums[j + 1] + nums[j + 2] > target\), 那么剩下的两个数无论如何取值, 四数之和一定大于 target, 退出第二重循环.

    class Solution:
    def fourSum(self, nums: List[int], target: int) -> List[List[int]]:
        # 排序
        nums.sort()
        results = []

        for i in range(len(nums) - 3):

            # 去重
            if i > 0 and nums[i] == nums[i - 1]:
                continue
            if nums[i] + nums[-1] + nums[-1] + nums[-2] < target:
                continue
            if nums[i] + nums[i + 1] + nums[i + 2] + nums[i + 3] > target:
                break

            for j in range(i + 1, len(nums) - 2):
                # 去重
                if j > i + 1 and nums[j] == nums[j - 1]: # 注意 j > i + 1 这个条件
                    continue
                if nums[i] + nums[j] + nums[-1] + nums[-2] < target:
                    continue
                if nums[i] + nums[j] + nums[j + 1] + nums[j + 2] > target:
                    break

                left, right = j + 1, len(nums) - 1
                while left < right:
                    sum_ = nums[i] + nums[j] + nums[left] + nums[right]
                    if sum_ < target:
                        left += 1
                    elif sum_ > target:
                        right -= 1
                    else:
                        results.append([nums[i], nums[j], nums[left], nums[right]])
                        while left < right and nums[left] == nums[left + 1]:
                            left += 1
                        while left < right and nums[right] == nums[right - 1]:
                            right -= 1
                        left += 1
                        right -= 1
        return results

    31.下一个排列#

    题目: 整数数组的一个 排列  就是将其所有成员以序列或线性顺序排列。整数数组的 下一个排列 是指其整数的下一个字典序更大的排列。更正式地，如果数组的所有排列根据其字典顺序从小到大排列在一个容器中，那么数组的 下一个排列 就是在这个有序容器中排在它后面的那个排列。如果不存在下一个更大的排列，那么这个数组必须重排为字典序最小的排列（即，其元素按升序排列）。例如

    • arr = [1,2,3] 的下一个排列是 [1,3,2]。
    • arr = [2,3,1] 的下一个排列是 [3,1,2]。
    • arr = [3,2,1] 的下一个排列是 [1,2,3]，因为 [3,2,1] 不存在一个字典序更大的排列。

    给你一个整数数组 nums ，找出 nums 的下一个排列。必须 原地 修改，只允许使用额外常数空间。

    class Solution:
    def nextPermutation(self, nums: List[int]) -> None:
        """
        Do not return anything, modify nums in-place instead.
        """
        i = len(nums) - 2
        # Step I: 从右向左, 找到第一个 nums[i] < nums[i + 1] 的元素
        while i > 0 and nums[i] >= nums[i + 1]:
            i -= 1

        # Corner Case: 如果整个数组为降序排列, 那么其下一个排列应该是升序排列
        if i == 0 and nums[i] > nums[i + 1]:
            nums.sort()
            return nums

        # Step II: 找到 i~end 中第一个大于 nums[i]的数, 并交换
        for k in range(len(nums) - 1, i, -1):
            if nums[k] > nums[i]:
                break
        nums[i], nums[k] = nums[k], nums[i]

        # Step III: nums[i] 之后的数升序排列
        # nums[i + 1:] = sorted(nums[i + 1:])
        left, right = i + 1, len(nums) - 1
        while left < right:
            nums[left], nums[right] = nums[right], nums[left]
            left += 1
            right -= 1
        return nums

    42.接雨水#

    题目: 给定 n 个非负整数表示每个宽度为 1 的柱子的高度图，计算按此排列的柱子，下雨之后能接多少雨水。

    方法1: 动态规划#

    对于索引 \(i\), 水能达到的最大高度为 \(i\) 两侧最大高度的最小值, 能接的雨水量为 \(i\) 处水能达到的最大高度减去 \(height[i]\)。据此可采用动态规划的思路: 创建两个数组 \(left\_max[\cdot]\) 和 \(right\_max[\cdot]\):

    • \(left\_max[i]\): 表示下标 \(i\) 及其左侧的位置的最大高度;
    • \(right\_max[i]\): 表示下标 \(i\) 及其右侧的最大高度.

    那么在 \(i\) 处能接的雨水量为 \(min(left\_max[i], right\_max[i]) - height[i]\).

    时间复杂度: \(O(n)\), 空间复杂度 \(O(n)\).

    方法2: 双指针#

    方法2是针对方法1的优化. 维护两个指针 \(left\), \(right\) 从两端向中间靠拢，维护两个变量 \(max_{left}\) 和 \(max_{right}\) 分别存储从左向右和从右向左的最大值。

    当两个指针未相遇时,

    • 若 \(height[left] < height[right]\), 必有 \(max_{left} < max_{right}\), \(left\) 处雨水量为 \(max_{left} - height[left]\);
    • 若 \(height[left] \geq height[right]\), 必有 \(max_{left} \geq max_{right}\), \(right\) 处雨水量为 \(max_{right} - height[right]\).

    时间复杂度: \(O(n)\), 空间复杂度 \(O(1)\).

    方法3: 单调栈#

    单调栈适用场景: 求解 第一个大于 xxx 或者 第一个小于 xxx. 方法论部分可以参考 单调栈

    def trap(self, height: List[int]) -> int:
    # 从左侧计算最大高度
    left_max = [0] * len(height)
    left_max[0] = height[0]
    for i in range(1, len(height)):
        left_max[i] = max(height[i], left_max[i - 1])

    # 从右侧计算最大高度
    right_max = [0] * len(height)
    right_max[-1] = height[-1]
    for i in range(len(height) - 2, -1, -1):
        right_max[i] = max(height[i], right_max[i + 1])

    # 计算 i 处搜集的雨水并累加到最终结果
    res = 0
    for i in range(len(height)):
        res += min(left_max[i], right_max[i]) - height[i]
    return res

    def trap(self, height: List[int]) -> int:
    left, right = 0, len(height) - 1
    left_max, right_max = 0, 0

    ans = 0
    while left < right:
        left_max = max(left_max, height[left])
        right_max = max(right_max, height[right])
        if height[left] < height[right]:
            ans += left_max - height[left]
            left += 1
        else:
            ans += right_max - height[right]
            right -= 1
    return ans

    def trap(self, height: List[int]) -> int:
    stack = [0]  # 保存索引
    ans = 0
    for i in range(1, len(height)):
        while len(stack) and height[i] > height[stack[-1]]:
            top = stack.pop()
            if not stack:
                break
            left = stack[-1]
            # 计算区域的宽度和高度
            w = i - left - 1
            h = min(height[left], height[i]) - height[top]
            ans += w * h
        stack.append(i)
    return ans
